# Domino Game server

This is a system containing both HTTP and WebSocket servers.

## Setup

**Install packages:**
```bash
npm i
```

**Run tests:**
```bash
npm test
```

**Run tests with coverage:**
```bash
npm test -- --coverage
```

**To start the server create .env file like .env.example (optionally) and run:**
```bash
npm start
```

**Build the project:**
```bash
npm run build
```
